## 🔧 技术特点

- **前端框架**: React 18
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **响应式设计**: 完美适配桌面和移动设备
- **现代化UI**: 渐变背景、毛玻璃效果、动画交互

## 📱 响应式支持

- 桌面端 (1200px+)
- 平板端 (768px - 1199px)
